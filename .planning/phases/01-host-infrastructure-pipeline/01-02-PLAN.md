<objective>
Add ClipCursor enforcement to the host: compute the allowed cursor rectangle excluding Display 3, apply it at startup, and automatically re-apply after session unlock and display change events via WTS session notifications and a QAbstractNativeEventFilter.

Purpose: HOST-04 is a HIGH-severity risk. Windows clears ClipCursor on every input desktop switch (Win+L, sleep/wake, UAC). Without WTS recovery, cursor lockout silently breaks after the first lock/unlock cycle.
Output: Cursor is confined to primary monitors at startup. After Win+L unlock and WM_DISPLAYCHANGE, cursor lockout is automatically restored.
</objective>

       def apply_clip_cursor(left: int, top: int, right: int, bottom: int) -> None:

    5. Add compute_allowed_rect:
       ```python
       def compute_allowed_rect(excluded_screen, all_screens) -> RECT:
           combined = None
           for screen in all_screens:
               if screen is excluded_screen:
                   continue
               geo = screen.geometry()
               combined = geo if combined is None else combined.united(geo)
           if combined is None:
               return RECT(0, 0, 1920, 1080)
           # QRect.right() returns left+width-1; use left()+width() for actual right edge
           return RECT(
               combined.left(),
               combined.top(),
               combined.left() + combined.width(),
               combined.top() + combined.height(),
           )
       ```
       CRITICAL: Use `left() + width()` not `right()` because Qt's `QRect.right()` returns `left + width - 1` (off by one). Same for bottom: use `top() + height()` not `bottom()`.

    6. Add WTS registration:
       ```python
       def register_session_notifications(hwnd: int) -> None:
           win32ts.WTSRegisterSessionNotification(hwnd, win32ts.NOTIFY_FOR_THIS_SESSION)

       def unregister_session_notifications(hwnd: int) -> None:
           win32ts.WTSUnRegisterSessionNotification(hwnd)
       ```

    7. Add Win32MessageFilter:
       ```python
       class Win32MessageFilter(QAbstractNativeEventFilter):
           def __init__(self, on_clip_needed):
               super().__init__()
               self._on_clip_needed = on_clip_needed

           def nativeEventFilter(self, event_type, message):
               if event_type == b"windows_generic_MSG":
                   msg = ctypes.wintypes.MSG.from_address(message.__int__())
                   if msg.message == WM_WTSSESSION_CHANGE:
                       if msg.wParam == WTS_SESSION_UNLOCK:
                           self._on_clip_needed()
                   elif msg.message == win32con.WM_DISPLAYCHANGE:
                       self._on_clip_needed()
               return False, 0
       ```
       MUST use `b"windows_generic_MSG"` (bytes), NOT `"windows_generic_MSG"` (str).
