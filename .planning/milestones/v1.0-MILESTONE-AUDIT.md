### Gap 1 — Phase 1 Missing VERIFICATION.md (Documentation, not functional)

**Options:**
1. Accept as tech debt — note in MILESTONES.md Known Gaps; proceed to complete-milestone
2. Retroactively run `/gsd:verify-work` for Phase 1 to create the missing document

### Gap 2 — IPC-03 Spec Deviation (Deliberate design, not a bug)

**Severity:** Low (spec/implementation misalignment)

`ProcessManager.stop_widget` in `host/process_manager.py` terminates widget processes with `proc.terminate()`. The IPC-03 requirement specifies "drains queue fully before join(); kill fallback after 5s". The implementation intentionally omits `join()` because it would deadlock the Qt main thread on Windows (the queue pipe is not drained while the main thread blocks on `join()`).

**Options:**
1. Accept as tech debt — amend IPC-03 language to match the implemented design in v1.x
2. Update test name `test_stop_widget_drains_queue_before_join` to reflect actual behavior
