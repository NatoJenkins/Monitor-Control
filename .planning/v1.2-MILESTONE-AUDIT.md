## Tech Debt

### MISS-01 — `_update_widget_settings` silent drop (Low Severity)

**File:** `control_panel/main_window.py` lines 408–416
**Issue:** Docstring says "Creates widget entry if absent" but code explicitly does not. If config.json `widgets` array is ever missing a `pomodoro` or `calendar` entry, all color values silently fail to persist on Save.
**Currently masked by:** config.json always contains all three widget entries in practice.
**Fix options:** Correct the docstring to match behavior, OR implement the create path.
**Affected requirements:** POMO-06, CAL-06 (edge case only)

### Cosmetic — `_text_color` vs `date_color` naming in CalendarWidget

**File:** `widgets/calendar/widget.py`
**Issue:** The `date_color` config key maps to `self._text_color` attribute (asymmetric with `_time_color`).
**Impact:** Functionally correct; naming inconsistency could cause confusion in future maintenance.
