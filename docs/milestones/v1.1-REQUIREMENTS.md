# Requirements: MonitorControl

**Defined:** 2026-03-27
**Core Value:** Keep productivity tooling off the primary monitors — widgets run persistently in a dedicated display the cursor cannot enter, requiring zero window management from the user.

## v1.1 Requirements

Requirements for v1.1 Startup & Distribution milestone. Each maps to roadmap phases.

### Autostart

- [x] **STRT-01**: User can enable host autostart at Windows login from the control panel Startup tab
- [x] **STRT-02**: User can disable host autostart from the control panel Startup tab
- [x] **STRT-03**: Autostart toggle reflects live HKCU registry state (reads current entry on every panel open, not cached)
- [x] **STRT-04**: Host launches without a terminal or console window when autostarted
- [x] **STRT-05**: Control panel Startup tab shows status label when autostart is active ("MonitorControl will start automatically at next login")

### Infrastructure

- [x] **INFRA-01**: Host and control panel resolve config.json relative to script/executable directory (not process cwd) — prerequisite for autostart and packaging both working correctly
- [x] **INFRA-02**: Host does not crash when launched without a console window (print() calls null-guarded for pythonw.exe context)

### Packaging

- [x] **PKG-01**: Control panel packaged as standalone .exe (no Python environment required)
- [x] **PKG-02**: Control panel .exe launches without a terminal or console window
- [x] **PKG-03**: Control panel .exe displays application icon (.ico, distinguishes from generic Python in Explorer/Task Manager)
- [x] **PKG-04**: PyInstaller build is reproducible via a committed .spec file

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Packaging (Host)

- **HPKG-01**: Host packaged as standalone .exe (no Python environment required)
- **HPKG-02**: Autostart entry points to host.exe directly (once host is packaged)
- **INFRA-03**: multiprocessing.freeze_support() called before set_start_method() in host (prerequisite for host .exe without spawn bomb)

### Polish (Backlog)

- **PLSH-01**: Pomodoro plays an audio cue at each phase transition
- **PLSH-02**: Widget crash detection with visual placeholder and control panel restart button
- **PLSH-03**: Notification widget shows scrollable queue of multiple simultaneous toasts
- **PLSH-04**: Calendar widget seconds display toggle

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Host .exe packaging | winrt-* 3.2.1 has no community PyInstaller hooks; expect 2-3 iteration cycles; deferred to v1.2 |
| Installer (NSIS/Inno Setup) | Single-user personal tool; no need for Start Menu entries or file associations |
| Task Scheduler autostart | HKCU Run key is simpler, appears in Windows Settings Startup apps, no invisible-window BLOCKER risk |
| Autostart for non-admin users requiring elevation | ClipCursor() works at user privilege; no elevation needed |
