    - `ColorPickerWidget(QWidget)` class:
      - Class attribute: `SATURATION = 0.8`
      - Class signal: `color_changed = pyqtSignal(str)`
      - `__init__`: `self._hue = 0.0`, `self._lightness = 0.5`, `self._updating = False`, calls `self._build_ui()` and `self._connect_signals()`
      - `_build_ui()`: Creates QVBoxLayout with three rows: (1) "Hue" QLabel + QSlider range 0-359 horizontal, (2) "Intensity" QLabel + QSlider range 0-100 horizontal, (3) _ColorSwatch + QLineEdit with QRegularExpressionValidator(QRegularExpression(r"#[0-9A-Fa-f]{0,6}")) and maxLength=7
      - `_connect_signals()`: hue_slider.valueChanged -> _on_hue_slider_changed, lightness_slider.valueChanged -> _on_lightness_slider_changed, hue_slider.sliderReleased -> _emit_color_changed, lightness_slider.sliderReleased -> _emit_color_changed, hex_field.editingFinished -> _on_hex_editing_finished
      - `_on_hue_slider_changed(value)`: if _updating return; set _hue = value/360.0; call _sync_swatch_and_hex()
      - `_on_lightness_slider_changed(value)`: if _updating return; set _lightness = value/100.0; call _sync_swatch_and_hex()
      - `_sync_swatch_and_hex()`: update swatch color and hex field text from _current_qcolor(), wrap hex_field.setText in _updating guard
      - `_on_hex_editing_finished()`: if _updating return; get text; create QColor(text); if not valid or not QColor.isValidColorName(text) return; update _hue only if hslHueF() >= 0.0; set _lightness from lightnessF(); call _sync_all_from_state(); emit color_changed
      - `_sync_all_from_state()`: set _updating=True; update all sliders, hex field, swatch; set _updating=False in finally block
      - `_emit_color_changed()`: if not _updating: emit color_changed with _current_qcolor().name()
      - `color()`: return _current_qcolor().name()
      - `set_color(hex_str)`: QColor(hex_str); if not valid return; update _hue only if hslHueF() >= 0.0; set _lightness; call _sync_all_from_state(). Does NOT emit.
      - `_current_qcolor()`: return QColor.fromHslF(self._hue, self.SATURATION, self._lightness)
